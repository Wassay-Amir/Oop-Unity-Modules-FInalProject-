# Switch Module

## Description
A simple on/off switch for circuits with visual feedback.

## Features
- Click to toggle ON/OFF
- Lever rotates smoothly
- Color changes (yellow/red)
- Implements IElectricalComponent interface

## How to Use
1. Drag Switch prefab into scene
2. Attach Switch.cs script
3. Click to toggle
4. Connect to Circuit manager

## Files Included
- Switch.cs
- Switch.prefab
- Materials/

## What Works
- Toggle functionality
- Visual feedback
- Rotation animation
- Circuit integration

## What's Missing
- Sound effects
- Advanced animations
