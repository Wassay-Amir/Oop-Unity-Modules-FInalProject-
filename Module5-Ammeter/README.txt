Description:
This prefab represents a digital ammeter designed for Unity-based electrical circuit simulations. It measures current flowing through a circuit and displays the value on a digital screen.

Included Components:
- Ammeter.prefab
- Ammeter.cs
- Terminal.cs
- ManualUI.cs
- 3D Body Model
- Positive Terminal (+)
- Negative Terminal (-)
- Digital Display
- Help/Instruction Button
- Instruction Panel

How It Works:
The ammeter reads the current value provided to its terminal and displays the current in amperes.

Usage:
1. Place the Ammeter prefab into the scene.
2. Connect the ammeter in series with the circuit.
3. Ensure the circuit simulation updates the current value.
4. Run the simulation.

Expected Input:
PositiveTerminal.Current

Output:
Displayed Current Reading (A)

Included Features:
- Digital current display
- Reusable prefab
- Positive and negative terminals
- Instruction manual popup
- Compatible with external wire systems

Not Included:
- Wire connection system
- Circuit solver
- Current calculation engine
- Automatic series connection detection
- Snap-to-wire functionality
- Electrical physics engine
